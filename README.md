# Portfolio Website

This is a personal and responsive **Portfolio Website** built using**HTML**,**CSS**, and **JavaScript**. It showcases my skills, projects, and contact information in a clean and professional layout.

## Features
- Responsive design for all devices  
- About Me section  
- Skills section  
- Projects showcase  
- Contact section  
- Resume download button  
- Smooth scrolling and simple animations  

## Technologies Used
- HTML  
- CSS  
- JavaScript
